# Trello Feature Listing (Comprehensive)

## 1. Core Board & Task Management

### Boards

* Unlimited boards (plan-based)
* Public / Private / Workspace visibility
* Board templates
* Starred boards
* Background customization

### Lists

* Vertical Kanban columns
* Drag-and-drop ordering
* Archive lists

### Cards

* Create / Edit / Delete
* Drag-and-drop movement
* Card templates
* Card duplication
* Archiving
* Watch functionality
